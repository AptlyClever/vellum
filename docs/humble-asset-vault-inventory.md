## License and handling boundaries

Observed general asset-marketplace constraints suggest the asset intake protocol should require license capture before use.

Unity Asset Store terms state that some assets may be governed by the standard Unity Asset Store EULA while others may have separate provider EULAs. The same terms restrict use of Unity Asset Store assets for AI or machine-learning training without express consent.

Fab's Standard License summary states that assets may be used privately or commercially, modified into projects, and distributed as part of projects, but not resold or redistributed as standalone assets.

Control Alt Games asset intake should therefore require:

- per-asset source capture,
- per-asset license/EULA note,
- no standalone redistribution,
- no public raw-asset repository commits,
- no AI/ML training usage unless the asset's license expressly permits it,
- derived renders/clips/symbol sheets stored separately from raw assets,
- clear project linkage for any derived asset.

## New project sparks captured

### The Door That Opens Incorrectly

A tiny first-person Threshold Affairs prototype. The player walks an institutional/commercial corridor; Door 2C opens into different impossible spaces depending on ruling state. No combat is required. The point is atmosphere, interaction, threshold fiction, and evidence collection.

Confirmed useful packs include Japanese Old Shopping Mall, Motel Room, Motel Reception, Cyberpunk Hospital/Clinic, Portal VFX Enhanced, Glass Bundle Material, Dirty Wall Material, and Industrial Warehouse Night Environment - GARAGE.

### Dobsonian Institute Evidence Generator

A tool/workflow that uses asset packs to create case-file visuals:

1. Pick a case type.
2. Pick a scene pack.
3. Place props.
4. Render three to five evidence images.
5. Export notes and images into a Threshold Affairs case-material format.

### Field Command Tabletop Mode

A companion visualizer, not a replacement for the current game. Field Command is represented as an institutional war table with unit markers, an Anchor zone, hostile pressure tracks, Field Authority counter, and incident lane.

Confirmed useful packs include Vertical Warehouse, HANGAR-X, Container City, Dungeon Ruins, Oil Rig Liope, Portal VFX Enhanced, Magic Projectiles, Magic Abilities, and Niagara Mega Pack Vol. 3.

### Control Alt Games Attract Mode

- Slots fake jackpot moment,

- Hail effects,
- LCARD room status,

Confirmed useful packs include Steampunk Zeppelin Station, HANGAR-X, Fireworks, Portal VFX Enhanced, Niagara Mega Pack Vol. 3, Stylized VFX - Water, Explosion VFX 2, and Toon Abilities.

### Asset Jam protocol

A recurring lightweight practice pattern:

1. Pick one asset pack.
2. Make one tiny scene.
3. Produce one screenshot, short clip, or playable interaction.
4. Record one Praxis Signal/Readout.
5. Do not overbuild.

This is intended to prevent the bundle from becoming unused shelfware.
